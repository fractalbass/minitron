Minitron

Minitron is an IoT project that demonstrates the use of web applications (Heroku) used in conjunction with Arduino and ESP8266 technologies. For more info on the project, please refer to this blog post.



All Under One Roof.

To make your own minitron, you will need the following:

  • Arduino Uno R3 or similar
  • Sparkfun ESP8266 WiFi Shield
  • Adafruit 9x16 Matrix Display (Charlieplexed)
  • Adafruit I31FL3731 Display Driver Board
  • One momentary push button of some type
  • One 2-KOhm Resister (red-black-red-gold)
  • Various jumper wires
  • (Recommended)Breadboard
